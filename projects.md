## Социальная сеть Л.Н. Толстого (2025–2026)

**Что это?**  
Цифровая энциклопедия связей ~10 000 имён из 90-томного собрания сочинений Л.Н. Толстого.

**Что сделали?**  
Разметка по TEI, построение онтологической модели, создание графа знаний, разработка интерактивной карты и визуализации социальной сети.

**Результат**  
Исследовательский инструмент и публичный нарратив, объединяющий переписку, географию и культурные связи эпохи.

## Архив А. Вознесенского: визуальная поэзия и сеть референций

**Что это?**  
R&D-пилот по цифровому представлению архива поэта-новатора Андрея Вознесенского, включая его визуальную поэзию и культурные отсылки.

**Что сделали?**  
Разработка TEI-шаблонов для визуальной поэзии, сценарии цифрового чтения, построение графа культурных референций, прототип интерфейса.

**Результат**  
Прототип исследовательской среды для работы с нестандартными поэтическими формами.

## Онтология ICON для гравюр ГМИИ им. Пушкина

**Что это?**  
Применение международной онтологии ICONCLASS к описанию аллегорических гравюр «Времена года» из коллекции Государственного музея изобразительных искусств им. А.С. Пушкина.

**Что сделали?**  
Адаптация ICON под русскоязычный контекст, семантическое обогащение метаданных, создание демо-навигации по коллекции.

**Результат**  
Схема онтологического описания и интерактивный прототип, демонстрирующий возможности linked data для музейных коллекций.
