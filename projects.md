## Проект "Слово Толстого": путеводитель по наследию писателя
*В процессе реализации (2025–2026)*

**О проекте**  
Цифровая энциклопедия, объединяющая тексты, биографические данные и визуальные материалы из наследия Л.Н. Толстого.

**Наша роль**  
- Работа с массивом данных из 90-томного собрания сочинений
- Разметка текстов по стандартам TEI
- Анализ фотоархива методами компьютерного зрения
- Извлечение структурированных данных с помощью ИИ

**Компетенции**  
- TEI 
- Компьютерное зрение 
- NLP 
- Структурирование данных

**Ссылки**  
[Сайт проекта](https://slovotolstogo.ru) · [GitHub](https://github.com/tolstoydigital)

## Архив Андрея Вознесенского: подходы к цифровому представлению визуальной поэзии
*Исследовательский проект*

**О проекте**  
Разработка методологии для цифрового представления экспериментальной поэзии и архивных материалов поэта-новатора.

**Наша роль**  
- Создание TEI-шаблонов для визуальной поэзии
- Разработка читательских сценариев для адаптированного цифрового издания
- Построение графа социальной сети поэта
- Анализ культурных референций в текстах
- Прототипирование интерфейса для нестандартных поэтических форм


**Компетенции**  
- TEI для визуальной поэзии
- Сетевой анализ UX для культурных проектов
- Цифровая эдиция
